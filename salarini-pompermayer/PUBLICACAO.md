# Publicacao da pagina de evidencias

O arquivo `evidencias/index.html` pode ser aberto localmente no Chrome ou publicado depois como site estatico.

## Opcao local

Abrir no Chrome:

```text
C:\Users\Frank\Documents\Main Project\familia-genealogia\evidencias\index.html
```

Esta e a opcao mais segura enquanto a pesquisa tiver dados familiares sensiveis.

## Opcao hospedada

Para publicar na web, preferir acesso privado:

- GitHub Pages privado, se disponivel na conta/plano;
- Cloudflare Pages com Cloudflare Access;
- Netlify ou Vercel com protecao de acesso;
- Google Drive/Docs apenas para visualizacao, nao como fonte principal versionada.

## Regra antes de publicar

Antes de subir publicamente:

1. revisar se ha dados de pessoas vivas;
2. revisar restricoes das fontes;
3. manter links e creditos;
4. publicar apenas imagens/recortes permitidos;
5. manter o repositorio privado como fonte de trabalho.
