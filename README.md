# Arvore genealogica - evidencias publicas

Site publico (GitHub Pages): https://msalarini.github.io/familia-genealogia-evidencias/

Este repositorio publica apenas os arquivos "publicaveis" da pesquisa genealogica do usuario (nunca certidoes/dados sensiveis de pessoas vivas sem autorizacao). Fonte de trabalho privada: repositorio `familia-genealogia`.

## Estrutura (a partir de 2026-07-21)

- `index.html` - pagina principal: linha paterna do usuario (familia **Salarini** / Pompermayer / Vezzoni / Bissa), com as pastas de evidencia por pendencia (`IT-001/`, `BR-001/`, `VZ-001/`, etc.) e `DOC-001/` ao lado.
- `gardacho/` - linha materna do casal (familia **Gardacho**, esposa do usuario), **em pausa por enquanto**. Contem `index.html` proprio.

## Organizacao das pastas de evidencia

Uma subpasta por pendencia:

```text
IT-001/
IT-002/
BR-001/
```

Nome de arquivo recomendado:

```text
AAAA-MM-DD_fonte_assunto.ext
```

Exemplo:

```text
2026-07-03_nati-trentino_riccardo-pompermajer.png
```

## Regra

Todo arquivo publicado aqui deve estar registrado em `pesquisa/evidencias_salarini.md` (ou `pesquisa/evidencias_gardacho.md`, para a pasta `gardacho/`) no repositorio privado.

Nao publicar dados sensiveis de pessoas vivas sem autorizacao explicita.
